# Словодел

Автономное Android-приложение для подсчёта очков в настольной игре «Словодел». Приложение работает локально на телефоне: без сервера, регистрации, рекламы, аналитики, Firebase, внешних баз данных и разрешения `INTERNET`.

## Возможности

- добавление и удаление игроков перед стартом партии;
- автоматический переход ходов и раундов;
- подсчёт стоимости русских слов с поддержкой буквы «ё»;
- буквенные бонусы для конкретного вхождения буквы;
- бонус слова ×2 или ×3;
- таблица текущих результатов;
- история ходов с бонусами и начисленными очками;
- отмена последнего обычного хода или пропуска;
- подтверждение завершения и сброса игры;
- итоговый рейтинг по очкам, затем по имени;
- восстановление незавершённой игры после перезапуска приложения.

## Структура проекта

- `app/src/main/java/ru/slovodel/scorekeeper/model` — модели игры;
- `app/src/main/java/ru/slovodel/scorekeeper/game` — подсчёт очков и логика ходов;
- `app/src/main/java/ru/slovodel/scorekeeper/storage` — локальное сохранение состояния;
- `app/src/main/java/ru/slovodel/scorekeeper/ui` — состояние интерфейса и Compose UI;
- `app/src/test/java/ru/slovodel/scorekeeper/game` — unit-тесты игровой логики;
- `.github/workflows/build-apk.yml` — сборка debug APK в GitHub Actions.

## Правила подсчёта

Стоимость букв: `а=1`, `б=3`, `в=2`, `г=3`, `д=2`, `е=1`, `ё=1`, `ж=7`, `з=4`, `и=1`, `й=5`, `к=2`, `л=2`, `м=2`, `н=2`, `о=1`, `п=2`, `р=2`, `с=2`, `т=2`, `у=3`, `ф=10`, `х=5`, `ц=8`, `ч=5`, `ш=8`, `щ=9`, `ъ=10`, `ы=4`, `ь=5`, `э=9`, `ю=8`, `я=3`.

Сначала применяются бонусы отдельных букв. Если в слове несколько одинаковых букв, бонус применяется только к выбранному номеру буквы. Затем итоговая сумма букв умножается на бонус слова.

Пример: `кот`, бонус `о ×2`, слово `×3`: `к=2`, `о=1×2`, `т=2`, сумма букв `6`, итог `18`.

## GitHub Actions

Workflow находится в `.github/workflows/build-apk.yml`.

Запуск вручную:

1. Откройте репозиторий на GitHub.
2. Перейдите во вкладку Actions.
3. Выберите workflow `Build debug APK`.
4. Нажмите `Run workflow`.

Также workflow запускается при `push` в `main` и при pull request в `main`. Он выполняет unit-тесты, собирает debug APK и загружает артефакт `slovodel-debug-apk`.

## Скачивание APK из Artifacts

1. Откройте завершённый запуск workflow.
2. Внизу страницы найдите блок Artifacts.
3. Скачайте `slovodel-debug-apk`.
4. В архиве будет файл `app-debug.apk`.

## Установка debug APK

Передайте `app-debug.apk` на Android-устройство и откройте файл. Android может показать предупреждение об установке приложения из неизвестного источника. Для личного тестирования нужно разрешить установку из выбранного приложения, например файлового менеджера или браузера.

Debug APK предназначен только для личного тестирования. Это не публикация в Google Play и не релизная сборка для распространения.
